‚öôÔ∏è enable()

caution

If the user's privilege level 15, the following command may not work correctly. Privilege level 15 user enters the enable mode directly when establish the connection. If you want to test this scenario, you must set the user's privilege level to 1.

‚Äã

from netmiko import ConnectHandler
device = {
"device_type":"cisco_ios",
"host":"192.168.10.12",
"username":"keremcan",
"password":"onemli",
"verbose":"True",
"secret":"onemli"}

‚Äã

Example#

with ConnectHandler(**device) as connection:
print(connection.find_prompt()) # --> to print the prompt before enable mode
connection.enable()
print(connection.find_prompt()) # --> to print the prompt after enable mode

Output

SSH connection established to 192.168.10.12:22
Interactive SSH session established
SW2>
SW2#

‚Äã

Possible Exceptions#

If you forget to give the secret password to the Cisco device, you will get an error like this:

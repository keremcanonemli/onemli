This method strip any backspace characters out of the output.. In this example I will use the output from send_command. And i accually added some backspace characters with code.

print(data)

with ConnectHandler(**device) as connection:
data=connection.send_command("show ip interface brief",strip_prompt=False)
data+="krmcn_onemli\x08\x08\x08\x08\x08\x08"
print(connection.strip_backspaces(data))

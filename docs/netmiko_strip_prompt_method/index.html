Example#

This method strips the trailing device prompt from the output. In this example I will use the output from send_command. send_command method has the strip_prompt parameter and I need to change the default value of this parameter.

Verify the prompt

with ConnectHandler(**device) as connection:
data=connection.send_command("show ip interface brief",strip_prompt=False)
print(data) # Print with prompt

Output

SSH connection established to 192.168.10.12:22
Interactive SSH session established

GigabitEthernet0/1 unassigned YES unset up up

Use the strip_prompt() method

with ConnectHandler(**device) as connection:
data=connection.send_command("show ip interface brief",strip_prompt=False)
print(connection.strip_prompt(data))

a_string#

This parameter that does not contain a default value. So, when you enter the command to send the parameters, it is assigned to the a_string and it strips the trailing device prompt from the output as in the example above.

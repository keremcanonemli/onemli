Export the Configuration Backup with Netmiko

In the previous post, we send the configuration to devices from the list and file. In this post, we will create a configuration backup file.

Device Definitions#

from netmiko import ConnectHandler
device1 = {
"device_type":"cisco_ios",
"host":"10.0.191.10",
"username":"admin",
"password":"C1SC0.."
}
device2 = {
"device_type":"cisco_ios",
"host":"10.0.192.20",
"username":"admin",
"password":"C1SC0.."
}

Get Configurations#

If we want to send the command to a device, the method we will use : send_command(). In cisco ios devices has two configuration files. The first one startup-config and the second one running-config. Let’s get them.

for device in (device1,device2):
with ConnectHandler(**device) as connection:
try:
running_config = connection.send_command("show running-config")
startup_config = connection.send_command("show startup-config")

File Operations#

Writing#

To create a new file in Python, we use the open() method. It takes two parameters. The first one is a file name and the second one is “w” parameter. “w” parameter will create a file if the specified file doesn’t exist. If it exists, python will overwrite the file with the “w” parameter. If you want to append the existing file use “a” parameter.

try:
with open("running_config.txt", "w") as file:
file.write(running_config)

Same operation for startup_config.txt file

with open("startup_config.txt", "w") as file:
file.write(startup_config)

Before the end we need to close except blocks

except Exception as e:
print("Houston We Have File Operation Problem")
except Exception as e:
print("Houston We Have A Connection Problem")

Date Stamp#

When we need to store configuration backup files with timestamp we need to import a new library:

from datetime import datetime

Stamp Format#

Get the current date and time information

NOW = datetime.now()

Format the date time information(More format codes)

STAMP=NOW.strftime("%Y_%m_%d")

Edit The Backup Files Names#

Edit startup_config.txt file name

with open(f"{STAMP}_running.txt", "w") as file:
file.write(startup_config)

Edit running_config.txt file name

with open(f"{STAMP}_startup.txt", "w") as file:
file.write(startup_config)

Hostname In Stamp#

If we want to add hostname information to our STAMP we need to access each device’s "host" key’s value. Actually, for-loop iterating all devices for us.

NOW = datetime.now()
STAMP = NOW.strftime("%Y_%m_%d")
for device in (device1,device2):
with ConnectHandler(**device) as connection:
try:
running_config = connection.send_command("show running-config")
startup_config = connection.send_command("show startup-config")
try:
with open(f"{STAMP}_running.txt", "w") as file:
file.write(startup_config)
with open(f"{STAMP}_startup.txt", "w") as file:
file.write(startup_config)
except Exception as e:
print("Houston We Have File Operation Problem")
except Exception as e:
print("Houston We Have A Connection Problem")

Change the 11th line with the following command:

with open(f"{device["host"]}_{STAMP}_running.txt", "w") as file:

And change the 13th line with the following command:

with open(f"{device["host"]}_{STAMP}_startup.txt", "w") as file:

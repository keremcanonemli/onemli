⚙️ exit_enable_mode()

caution

If the user's privilege level 1, the following command may not work correctly. Because before exiting enable mode, you must enter enable mode.

​

from netmiko import ConnectHandler
device = {
"device_type":"cisco_ios",
"host":"192.168.10.12",
"username":"keremcan",
"password":"onemli",
"verbose":"True",
"secret":"onemli"}

​

Example#

with ConnectHandler(**device) as connection:
print(connection.find_prompt())
connection.exit_enable_mode()
print(connection.find_prompt())

Output

SSH connection established to 192.168.10.12:22
Interactive SSH session established
SW2#
SW2>

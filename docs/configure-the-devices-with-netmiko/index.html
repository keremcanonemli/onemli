Configure the Devices with Netmiko

In the previous post, we connected to multiple hosts with different variations. In this post, we will make configurations using methods provided by netmiko.

Device Definition#

from netmiko import ConnectHandler
device= {
'device_type': 'cisco_ios',
'host': '10.0.191.10',
'username': 'admin',
'password': 'C1SC0..',
}
with ConnectHandler(**device) as connection:
...

send_config_set() Method#

If we want to send config to the device we need a special method and this is : send_config_set().Set is the key point in this method.

Set meaning, you can send one configuration command, or if you want you can send more than one configuration command. As you might guess config commands are iterable and executed one by one with the send_config_set() method. Another key point is send_config_set() will automatically enter and exit configuration mode.

Send Configuration Commands Line by Line#

#These commands are acceptable but not practical
connection.send_config_set("interface lo99")
connection.send_config_set("ip address 99.99.99.99 255.255.255.255")
connection.send_config_set("no shutdown")

Make Configuration Commands Iterable#

configuration = ["interface lo99", "ip address 99.99.99.99 255.255.255.255", "no shutdown"]

Send Iterable Commands to Device and Print What it Sends#

print(connection.send_config_set(configuration))

Output#

configure terminal

RTR(config)#interface lo99

RTR(config-if)#end

send_config_from_file() Method#

If we want to send config to the device from a file, as you might guess we will use this method:send_config_from_file(). The method reads and processes the file line by line. This command automatically enters and exits configuration mode.

Create a simple configuration file (e.g reset_loopback.txt)#

no interface loopback99

print(connection.send_command("show ip interface brief"))

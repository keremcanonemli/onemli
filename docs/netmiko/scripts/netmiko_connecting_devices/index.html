with Only For Loop#

The for loop is a useful option for connecting to multiple devices. Each time the loop returns, it will send the dictionaries we provide as arguments to ConnectHandler() one by one.

Example#

for device in (device1, device2, device3):
with ConnectHandler(**device) as netConnect:
print(netConnect.send_command("show running-config"))

If you want to handle errors with try-except:

for device in (device1,device2,device3):
try:
with ConnectHandler(**device) as netConnect:
print(netConnect.send_command("show running-config"))
except Exception as e:
print("Houston We Have a Problem")

with Lists#

The lists are useful for organizing and accessing multiple devices with a loop. Each time the loop returns, it will send the dictionary we provide as arguments to ConnectHandler() one by one.

Example#

devices_list = [device1,device2,device3]
for device in devices_list:
with ConnectHandler(**device) as connection:
output = connection.send_command("show ip interface brief")
print(output)

If you want to handle errors with try-except:

for device in devices_list:
with ConnectHandler(**device) as connection:
try:
output = connection.send_command("show ip interface brief")
print(output)
except Exception as e:
print("Houston We Have a Problem")

with Excel#

In my opinion, the most practical method for connecting devices with Netmiko is to use excel files. It provides flexibility in adding new devices and changing the information of previously added devices. This is just my opinion.

Excel File#

Example#

from netmiko import ConnectHandler
from openpyxl import load_workbook
import time
from datetime import datetime
# Worksheet from Hosts.xlsx file. Set the active worksheet index 0
workBook = load_workbook("Hosts.xlsx")
workBook.active = 0
# Sheet id's starting at 0 in openpyxl, like list's index value.
workSheet = workBook.active
# if there is any sleep value specified in Hosts.xlsx, we can assign value to SLEEP variable
SLEEP = 0
# min_row specifies from which line the loop will start
for row in workSheet.iter_rows(min_row=2):
# we will add values from excel file one by one to the DEVICE dictionary
DEVICE = {}
for cell in row:
if cell.column == 1 and str(cell.value).strip() is not None:
DEVICE["device_type"] = cell.value
elif cell.column == 2 and str(cell.value).strip() is not None:
DEVICE["host"] = cell.value
elif cell.column == 3 and str(cell.value).strip() is not None:
DEVICE["username"] = cell.value
elif cell.column == 4 and str(cell.value).strip() is not None:
DEVICE["password"] = cell.value
elif cell.column == 5 and str(cell.value).strip() is not None:
DEVICE["secret"] = cell.value
elif cell.column == 6 and str(cell.value).strip() is not None and str(cell.value).isdigit():
SLEEP = cell.value
try:
print(10 * "*", "START TIME --> ",datetime.now(),"CONNECTING --> ", DEVICE["host"], 10 * "*")
net_connect = ConnectHandler(**DEVICE)
time.sleep(SLEEP)
print(net_connect.send_command("show vlan brief"))
print(10 * "*", "FINISH TIME --> ",datetime.now(),"FINISHED --> ", DEVICE["host"], 10 * "*")
except Exception as e:
print(10 * "*", "FAILED TIME --> ",datetime.now(),"FAILED --> ", DEVICE["host"], 10 * "*")
print(f"Houston We Have a Connection Problem to {DEVICE['host']}")

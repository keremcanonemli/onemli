Introducing Network Automation with Netmiko

What is Netmiko ?#

Netmiko is the simplest, multi-functional, open-source, and multi-vendor SSH Python library. Which makes connecting to network devices via SSH a breeze. If youâ€™ve never used it before, donâ€™t worry. All you need is a computer with Python installed and an IDE where you can develop your code.

Installation#

The following commands allow installing the latest version of the netmiko library with pip. Netmiko needs more than one library to work properly and functionally. Some of these are pyserial, paramiko, scp, ntc-templates. But donâ€™t worry when you install the Netmiko others install automatically.

pip install netmiko

Verify (optional)#

pip freeze | findstr netmiko

Importing#

You can use any of the following import methods.

Method 1#

from netmiko import ConnectHandler

Method 2#

import netmiko

If you use second method, when you need to call ConnectHandler method you must use syntax like this : netmiko.ConnectHandler()

Connection Informations#

You can use many different variations to connect devices. These examples are the simplest variations.

Connect with Arguments#

from netmiko import ConnectHandler
connection = ConnectHandler(device_type='cisco_ios', host='10.0.20.21', username='admin', password='C1SC0..')

If you want to connect to a device other than cisco_ios, you need to change the value of device_type. The device_type values of the Vendor and its products are in /site-packages/netmiko/ssh_dispatcher.py file.

Connect with Dictionary#

from netmiko import ConnectHandler
device = {
'device_type': 'cisco_ios',
'host': '10.0.20.21',
'username': 'admin',
'password': 'C1SC0..',
}
connection = ConnectHandler(**device)

Establish the Connection#

You passed the device information to ConnectHandler and now, you need to establish the connection .In this example i used send_command() method. This method allows you to send the command you specified to the device.

connection.send_command("show running-config")

If you want to show deviceâ€™s response you need to write in the print function or transfer the data to new defined variable.

print(connection.send_command("show running-config"))

or

output = connection.send_command("show running-config")
print(output)

Close the Connection#

If you connect to the device with the above connection type, you must close the connection manually:

connection.close()

Automatic Disconnection#

with ConnectHandler(**device) as connection:
output=connection.send_command("show running-config")
print(output)

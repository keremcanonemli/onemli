command_string#

It is the only parameter that does not contain a default value. So, when you enter the command to send the parameters, it is assigned to the command_string.

# They give the same result
print(connection.send_command("show ip interface brief"))
print(connection.send_command(command_string="show ip interface brief"))

strip_prompt#

This parameter's default value is strip_prompt=True. Remove the trailing router prompt from the output.

# True is the default value. So you don't need to write. It was written here as an example.
print(connection.send_command("show ip interface brief",strip_prompt=True))

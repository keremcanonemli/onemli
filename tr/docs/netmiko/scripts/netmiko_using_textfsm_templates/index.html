💻 TextFSM Templates

NTC Templates

NTC-Templates is an Apache 2.0 licensed project of Network To Code. NTC-Templates contains a set of multi-vendor templates. In background uses Google’s TextFSM parsing engine. TextFSM templates for parsing show commands of network devices.

Installation#

NTC-Templates automatically installed with Netmiko. If you want to install without netmiko you just need following command :

pip install ntc_templates

​

Supported Show Commands#

TextFSM templates are multi-vendor and support many demonstration commands. Just click this link to view which commands it supports. Also, if you want to access this list from your computer:/site-packages/ntc_templates/templates

​

from netmiko import ConnectHandler
import json

Using Excel#

In the previous example, we connect devices with excel. In this example, I add new columns in the Hosts.xlsx for parsing with text_fsm formatting with JSON.

​

Example#

device_typehostusernamepasswordsecretsleeptext_fsmjson

cisco_ios192.168.10.13keremcanonemlionem.li10FALSEFALSE

​

from netmiko import ConnectHandler
from openpyxl import load_workbook
from datetime import datetime
import json
import time
# Worksheet from Hosts.xlsx file. Set the active worksheet index 0
workBook = load_workbook("Hosts.xlsx")
workBook.active = 0
# Sheet id's starting at 0 in openpyxl, like list's index value.
workSheet = workBook.active
# if there is any sleep value specified in Hosts.xlsx, we can assign value to SLEEP variable
SLEEP = 0
# min_row specifies from which line the loop will start
for row in workSheet.iter_rows(min_row=2):
# we will add values from excel file one by one to the DEVICE dictionary
DEVICE = {}
PARSING= {}
for cell in row:
if cell.column == 1 and str(cell.value).strip() is not None:
DEVICE["device_type"] = cell.value
elif cell.column == 2 and str(cell.value).strip() is not None:
DEVICE["host"] = cell.value
elif cell.column == 3 and str(cell.value).strip() is not None:
DEVICE["username"] = cell.value
elif cell.column == 4 and str(cell.value).strip() is not None:
DEVICE["password"] = cell.value
elif cell.column == 5 and str(cell.value).strip() is not None:
DEVICE["secret"] = cell.value
elif cell.column == 6 and str(cell.value).strip() is not None and str(cell.value).isdigit():
SLEEP = cell.value
# TextFSM Column value checking
elif cell.column == 7 and str(cell.value).strip() is not None:
if cell.value == "True" or cell.value == "False":
PARSING["textfsm"] = cell.value
else:
print("TextFSM Value Error")
# Json Column value checking
elif cell.column == 8 and str(cell.value).strip() is not None:
if cell.value == "True" or cell.value == "False":
PARSING["json"] = cell.value
else:
print("JSON Value Error")
# Establish the connection
try:
print(10 * "*", "START TIME --> ",datetime.now(),"CONNECTING --> ", DEVICE["host"], 10 * "*")
net_connect = ConnectHandler(**DEVICE)
if PARSING["json"] == "True":
time.sleep(SLEEP)
print(json.dumps(net_connect.send_command("show vlan brief",use_textfsm=PARSING["textfsm"]),indent=4))
print(10 * "*", "FINISH TIME --> ", datetime.now(), "FINISHED --> ", DEVICE["host"], 10 * "*","\n\n")
else:
time.sleep(SLEEP)
print(net_connect.send_command("show vlan brief",use_textfsm=PARSING["textfsm"]))
print(10 * "*", "FINISH TIME --> ",datetime.now(),"FINISHED --> ", DEVICE["host"], 10 * "*","\n\n")
except Exception as e:
print(10 * "*", "FAILED TIME --> ",datetime.now(),"FAILED --> ", DEVICE["host"], 10 * "*")
print(f"Houston We Have a Connection Problem to {DEVICE['host']}")
